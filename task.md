# 任务列表

## 网页功能
- 有比较简洁和现代风格的，浅色主题的界面（参考openai官网），页面尽可能使用中文
- 有输入框，能够输入自然语言，让ai自动生成日程计划。AI调用ChatGLM的GLM-zero
- ai生成的日程需要按照标准xml格式，与用户创建后的形式相同，储存在数据库中
- 用户也可以自行创建日程，创建页面需要简洁美观，创建结果以标准xml储存在数据库中
- 需要有一个数据库
- 需要实现账号服务，并且拥有相应的安全保障措施
- 可以使用IP地址访问该网页（暂无网址）
- 需要预防网络攻击
